# UX Proposals

## Improved Onboarding Flow [CRITICAL]
**Problem**: High drop-off rate during KYC and account setup
**Solution**: 5-step progressive onboarding with progress indicator, skip options, and save-and-continue
**Impact**: +25% onboarding completion rate

## Dark Mode Support [HIGH]
**Problem**: No dark mode — causes eye strain for night traders
**Solution**: System-aware dark mode with manual toggle, persisted in user preferences
**Impact**: +15% evening session engagement

## Full RTL Arabic Support [CRITICAL]
**Problem**: Arabic UI is partially RTL — charts and tables break in Arabic mode
**Solution**: Full RTL CSS with dir=rtl, RTL-aware chart labels, Arabic number formatting
**Impact**: +40% Arabic user retention

## Improved Empty States [MEDIUM]
**Problem**: Empty watchlist/portfolio shows blank page with no guidance
**Solution**: Illustrated empty states with clear CTAs and sample data preview
**Impact**: +20% feature discovery

## Global Search Upgrade [HIGH]
**Problem**: Search only matches exact ticker symbols — misses company names
**Solution**: Fuzzy search with company name, ticker, ISIN lookup + recent searches
**Impact**: +30% symbol discovery rate
